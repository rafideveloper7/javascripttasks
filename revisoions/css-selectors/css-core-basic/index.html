1. CSS Syntax

CSS rules consist of a selector and a declaration block:

selector {
  property: value;
}

Example:

h1 {
  color: blue;
  font-size: 2rem;
}

2. Selectors

Selectors target HTML elements to style them:

/* Type selector */
p { color: black; }

/* Class selector */
.highlight { background: yellow; }

/* ID selector */
#header { height: 100px; }

/* Attribute selector */
[data-example] { border: 1px solid; }

This paragraph has a data-example attribute.

This paragraph has an ID selector applied.

3. Properties & Values

Properties specify what to style, values specify how to style it:

background: linear-gradient(135deg, #3a86ff, #8338ec);
.element {
  width: 100%; /* property: value; */
  height: 200px;
  background: linear-gradient(...);
  color: white;
  display: flex;
  justify-content: center;
  align-items: center;
}

4. Comments (/* */)

Comments are ignored by the browser and used for documentation:

/* This is a single-line comment */

/*
 * This is a
 * multi-line comment
*/

/* Specific section styling */
.header {
  /* Temporary border for debugging */
  border: 1px solid red;
}

5. !important

The !important rule overrides all other declarations:

.normal-rule {
  color: red;
}

.important-rule {
  color: green !important;
}

This text is green due to !important, even though a later rule tries to make it red.

Note: !important should be used sparingly as it makes debugging difficult.
