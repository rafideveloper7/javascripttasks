Master CSS Selectors

Interactive playground to learn and practice all CSS selectors

Universal Selector

Selects all elements in the document.

* { margin: 0; padding: 0; }
This entire page uses the universal selector for basic reset styles.

Type Selector

Selects all elements of the given type.

h1 { color: blue; }

This paragraph is styled with a type selector.

Class Selector

Selects elements with the specified class.

.selector-card { background: white; }
These cards are all selected using their class.

ID Selector

Selects the element with the specified ID.

#main-header { background: blue; }
The page header is styled using its ID.

Descendant Selector

Selects elements that are descendants of another element.

.card p { color: gray; }

This paragraph is selected because it's inside a div with class "descendant-example".

Child Selector

Selects elements that are direct children of another element.

ul > li { list-style: none; }

I'm a direct child (styled)

I'm a grandchild (not styled)

Adjacent Sibling

Selects the element that is immediately after another element.

h2 + p { margin-top: 0; }

Target element

Immediately after (styled)

Not immediately after (not styled)

General Sibling

Selects all siblings that follow another element.

h2 ~ p { color: gray; }

Target element

Sibling after (styled)

Div in between

Another sibling after (styled)

Attribute Selector

Selects elements with the specified attribute.

[data-example] { border: 1px solid; }
This div has a data-attribute-example attribute.

Pseudo-classes

Selects elements in a specific state.

a:hover { color: red; }
  • :hover - Hover over me
  • :first-child - I'm first
  • :nth-child(2n) - I'm even
  • :last-child - I'm last

Pseudo-elements

Selects specific parts of an element.

p::first-line { font-weight: bold; }

This text has pseudo-elements before and after it.

:not Selector

Selects elements that don't match the selector.

p:not(.exclude) { color: blue; }

I'm selected by :not

I'm excluded

I'm also selected

:is Selector

Selects any element that matches any of the selectors.

:is(header, footer) { background: black; }
Header-like element
Regular content
Footer-like element

:has Selector

Selects elements that contain the specified selector.

div:has(p) { border: 1px solid; }

I'm a special child that styles my parent
